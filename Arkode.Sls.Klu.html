val solver : Arkode.Sls.sparse_jac_fn ->
       int -> [> Nvector_serial.kind ] Arkode.serial_linear_solver
A direct linear solver on sparse matrices. In the call, klu jfn nnz, jfn is a callback function that computes an approximation to the Jacobian matrix and nnz is the maximum number of nonzero entries in that matrix.

val reinit : [> Nvector_serial.kind ] Arkode.serial_session -> int -> int -> bool -> unit
Reinitializes the Jacobian matrix memory and flags. In the call, reinit s n nnz realloc, n is the number of system state variables, and nnz is the number of non-zeroes in the Jacobian matrix. New symbolic and numeric factorizations will be completed at the next solver step. If realloc is true, the Jacobian matrix will be reallocated based on nnz.
See sundials: ARKKLUReInit
