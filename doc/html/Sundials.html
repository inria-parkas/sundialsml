Miscellaneous utility functions


val print_time : string * string -> float -> unit
print_time (s1, s2) t prints t with s1 on the left and s2 on the right.
val extra_time_precision : bool Pervasives.ref
Controls the precision of Sundials.print_time.

If true the format %.15e is used, otherwise %e (the default) is used.

val format_float : string -> float -> string
format_float fmt f formats f according to the format string fmt, using the low-level caml_format_float function.
val floata : float -> string
Equivalent to format_float "%a".

floata f returns the bit-level representation of f in hexadecimal as a string.
