Module Cvode.Bandmatrix.Col


module Col: sig .. end
Access banded matrix columns
type c
This type represents a bandmatrix (DlsMat) column.

The underlying array cannot be exposed directly in Ocaml as a Bigarray because there is no simple way to attach a custom finalize function to such a big array.

See sundials: BAND_COL 
val get_col : Cvode.Bandmatrix.t -> int -> c
get_col a j returns the diagonal element of the j-th column of the n by n band matrix a, where 0 <= j <= n - 1. The resulting column may be indexed from -mu(a) to ml(a).
See sundials: BAND_COL 
val get : c -> int * int -> float
get c (i, j) returns the (i, j)th entry of the band matrix from which the column c has already been selected; provided that j - mu(c) <= i <= j + ml(c).
See sundials: BAND_COL_ELEM 
val set : c -> int * int -> float -> unit
set c (i, j) v stores the value v at the (i, j)th entry of the band matrix from which the column c has already been selected; provided that j - mu(c) <= i <= j + ml(c).
See sundials: BAND_COL_ELEM
