Module Cvode.Directdensematrix


module Directdensematrix: sig .. end
Operations for creating and manipulating direct dense matrices.
type t
This type represents a realtype ** returned from a call to Cvode.Directdensematrix.new_dense_mat.

The underlying array cannot be exposed directly in Ocaml as a Bigarray because it is an array of arrays (an lliffe vector) and, anyway, there is no simple way to attach a custom finalize function to such a big array.

See sundials: Small dense matrices
See sundials: newDenseMat

Basic access


val new_dense_mat : int * int -> t
new_dense_mat m n returns an m by n dense small matrix.
See sundials: newDenseMat 
val get : t -> int * int -> float
get a (i, j) returns the value at row i and column j in the m by n matrix a, where 0 <= i < m and 0 <= j < n.
val set : t -> int * int -> float -> unit
set a (i, j) v stores the value v at row i and column j in the m by n matrix a, where 0 <= i < m and 0 <= j < n.

Calculations


val copy : t -> t -> int * int -> unit
copy src dst (m, n) copies the contents of one m by n matrix into another.
See sundials: denseCopy 
val scale : float -> t -> int * int -> unit
val add_identity : t -> int -> unit
add_identity a n increments an n by n matrix by the identity matrix.
See sundials: denseAddIdentity 
val getrf : t -> int * int -> int_array -> unit
getrf a (m, n) p performs the LU factorization of an m by n matrix a with partial pivoting according to p.
Raises ZeroDiagonalElement Zero found in matrix diagonal
See sundials: denseGETRF 
val getrs : t -> int -> int_array -> real_array -> unit
getrs a n p b finds the solution of ax = b using LU factorization. a must be an n by n matrix.
See sundials: denseGETRS 
val potrf : t -> int -> unit
potrf a n performs the Cholesky factorization of a real symmetric positive n by n matrix.
See sundials: DensePOTRF
See sundials: densePOTRF 
val potrs : t -> int -> real_array -> unit
potrs a n b finds the solution of ax = b using Cholesky factorization. a must be an n by n matrix.
See sundials: densePOTRS 
val geqrf : t -> int * int -> real_array -> real_array -> unit
geqrf a (m, n) beta work performs the QR factorization of an m by n matrix, where m >= n.
See sundials: denseGEQRF 
val ormqr : a:t ->
       mn:int * int ->
       beta:real_array -> v:real_array -> w:real_array -> work:real_array -> unit
ormqr a beta v w work computes the product w = Qv, with Q calculated using Cvode.Directdensematrix.geqrf.
See sundials: denseORMQR
