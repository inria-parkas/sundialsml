Module Arkode.Spils.Banded

module Banded: sig .. end
Banded preconditioners.
val prec_left : Arkode.bandrange ->
       (Nvector_serial.data, [> Nvector_serial.kind ]) Arkode.Spils.preconditioner
A band matrix Arkode.Spils.preconditioner based on difference quotients. The call prec_left br instantiates a left preconditioner which generates a banded approximation to the Jacobian with br.mlower sub-diagonals and br.mupper super-diagonals.
See sundials: ARKBandPrecInit
val prec_right : Arkode.bandrange ->
       (Nvector_serial.data, [> Nvector_serial.kind ]) Arkode.Spils.preconditioner
Like Arkode.Spils.Banded.prec_left but preconditions from the right.
See sundials: ARKBandPrecInit
val prec_both : Arkode.bandrange ->
       (Nvector_serial.data, [> Nvector_serial.kind ]) Arkode.Spils.preconditioner
Like Arkode.Spils.Banded.prec_left but preconditions from both sides.
See sundials: ARKBandPrecInit

val get_num_rhs_evals : [> Nvector_serial.kind ] Arkode.serial_session -> int
Returns the number of calls to the right-hand side callback for the difference banded Jacobian approximation. This counter is only updated if the default difference quotient function is used.
