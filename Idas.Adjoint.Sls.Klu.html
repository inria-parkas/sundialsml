val reinit : [> Nvector_serial.kind ] Idas.Adjoint.serial_bsession ->
       int -> int -> bool -> unit
Reinitializes the Jacobian matrix memory and flags. In the call, reinit s n nnz realloc, n is the number of system state variables, and nnz is the number of non-zeroes in the Jacobian matrix. New symbolic and numeric factorizations will be completed at the next solver step. If realloc is true, the Jacobian matrix will be reallocated based on nnz.
